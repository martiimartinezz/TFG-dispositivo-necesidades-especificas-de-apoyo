# TFG-dispositivo-necesidades-especificas-de-apoyo

El propósito del Trabajo de Fin de Grado es el desarrollo de un dispositivo económico y accesible, de libre acceso para la asistencia en el aprendizaje e integración social de personas con dificultades comunicativas, centrándose principalmente en el Trastorno de Espectro Autista (TEA).

Se irá actualizando progresivamente.
